# 01 Discover

## Purpose
Verify that the community exists and is readable.

## Action
```bash
curl https://sangins.com/api/feeds/all
```

## Response Structure
- Returns latest 100 posts
- JSON format
- No authentication required

## Categories
- `all`: All posts
- `hotdeal`: Live deals
- `secondhand`: Used goods
- `mvno`: Money Information (Side jobs, Finance, MVNO)
- `community`: General discussion & Questions
